# User Stories — Todo App

> Format: "As a [role], I want [action] so that [benefit]."
> Status: ✅ Done | 🔲 Not yet implemented

## Navigation

- ✅ As a user, I want a home page that lists my projects, so that I can quickly find the app I want to use.
- ✅ As a user, I want a "back to all projects" link on the todo page, so that I can return to the home page without using the browser back button.

## Todo Management

- ✅ As a user, I want to type a todo title and click "Add Todo", so that I can add a new task to my list.
- ✅ As a user, I want to see my new todo appear in the list immediately after adding it, so that I get instant feedback without refreshing the page.
- ✅ As a user, I want to see a validation message if I submit an empty or blank title, so that I know what went wrong.
- ✅ As a user, I want the input to clear automatically after I add a todo, so that I can quickly add the next one.
- ✅ As a user, I want to delete a todo by clicking the trash icon, so that I can remove tasks I no longer need.
- ✅ As a user, I want the todo to disappear from the list immediately after deleting it, so that I get instant feedback.
- ✅ As a user, I want to see an empty state message when I have no todos, so that the page doesn't look broken.

## Feedback & States

- ✅ As a user, I want to see a loading spinner on the submit button while my todo is being saved, so that I know the app is working.
- ✅ As a user, I want to see a success message after adding a todo, so that I know it was saved.
- ✅ As a user, I want to see an error message if something goes wrong, so that I can try again.
- ✅ As a user, I want the delete button to show a spinner while deleting, so that I know the action is in progress.

## Appearance

- ✅ As a user, I want to switch between light and dark mode, so that I can use the app comfortably in any lighting.
- ✅ As a user, I want the app to remember my theme preference, so that I don't have to set it every time.

## Not yet implemented (P1)

- 🔲 As a user, I want to mark a todo as done with a checkbox, so that I can track my progress.
- 🔲 As a user, I want to edit a todo's title inline, so that I can correct mistakes without deleting and re-adding.
- 🔲 As a user, I want my todos to persist after I refresh the page, so that I don't lose my list.
- 🔲 As a user, I want to register and log in, so that my todos are private and not shared with others.
- 🔲 As a user, I want a confirmation before deleting a todo, so that I don't accidentally remove the wrong task.
